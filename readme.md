# Emulator for CHIP8

## How to use

```
usage: chip8.py [-h] [-t {console,pygame}] [-f FILE] [-d | --debug | --no-debug]

CHIP-8 Emulator

options:
  -h, --help            show this help message and exit
  -t {console,pygame}, --type {console,pygame}
                        Type of emulator. By default: pygame.
  -f FILE, --file FILE  Path to ROM file.
  -d, --debug, --no-debug
```
