# Emulator for CHIP8

## How to use

```
usage: chip8.py [-h] [-t {console,pygame}] [-r FILE]

options:
  -h, --help            show this help message and exit
  -t {console,pygame}, --type {console,pygame}
                        Type of emulator. By default: pygame.
  -r FILE, --rom FILE   Path to ROM file.
```
